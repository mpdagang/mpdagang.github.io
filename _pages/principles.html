Principles for Reflective Web Archiving src

  1. Attempt to save as much material as possible
  2. Performed by archivists and/or institutions outside the community
  3. Focused on the universal over the personal
  4. Concerned with preservation and restoration rather than recirculation

Evans' Law of Web Archiving src

Stewart's Principles of Precision src

  1. Complexity often masquerades as sophistication; simplicity requires confidence.
  2. True mastery lies not in conquering all tools, but in wielding the fewest with precision.
  3. The server that renders HTML has solved more problems than the framework that promises to.
  4. Each abstraction should simplify something; when it doesn’t, we’ve abstracted too soon.

Verou's Principle of User Effort src

In the economy of user effort, be a bargain, not a scam. Treat user effort as a currency. To create a product users love, design the tradeoff curve of use case complexity to user effort with the same care you design your pricing scheme.
